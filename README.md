## 项目说明
### **重要提示**：
1： **没有数据库，数据均为文本化的数据，增删改查均为对文本数据的操作，之后可能会有数据库**
### **目录说明**
build： 编译后的服务器代码

client： 打包后后的项目

mockData： 模拟数据 

 1. userpassword.json： 用户名及密码
 2. user.json:用户的具体信息
 3. loginlog.json：操作日志
 4. adminuser.json：员工的信息
 5. adminuerdetail.json： 员工的具体的信息
 
model： 一些所以需要的函数
